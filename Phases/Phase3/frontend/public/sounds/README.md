# Alarm Sound

Please add an alarm sound file named `alarm.mp3` to this directory.

## Where to get free alarm sounds:

1. **Freesound.org** - https://freesound.org/search/?q=alarm+clock
2. **Zapsplat** - https://www.zapsplat.com/sound-effect-category/alarms-and-sirens/
3. **Pixabay** - https://pixabay.com/sound-effects/search/alarm/

## Requirements:

- File name: `alarm.mp3`
- Format: MP3
- Duration: 3-10 seconds (will loop automatically)
- Volume: Medium (not too loud)

## Quick Option:

Search YouTube for "alarm clock sound effect", download using a YouTube to MP3 converter, and save as `alarm.mp3` in this folder.

The app will work without the sound file, but notifications won't have audio.
